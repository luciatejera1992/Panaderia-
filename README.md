# Panadería

## Descripción del Proyecto

El proyecto **Panadería** tiene como objetivo analizar y visualizar los datos de ventas y precios de productos de una panadería real. A partir de este análisis, se busca desarrollar un sistema accionable que permita optimizar las operaciones, mejorar la toma de decisiones y maximizar los ingresos.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- **`PANADERIA/EDA.ipynb`**: Un cuaderno Jupyter que contiene el análisis exploratorio de datos (EDA). En este archivo se realizan tareas como:
  - Limpieza y transformación de datos.
  - Análisis de valores nulos y duplicados.
  - Visualización de patrones de ventas y precios.
  - Integración de los datasets de ventas y precios.
  - Identificación de productos sin precio y análisis de su impacto.

- **`PANADERIA/DATA/Bakerysales.csv`**: Dataset que contiene los registros de ventas, incluyendo:
  - Fecha y hora de la transacción.
  - Día de la semana.
  - Total de la venta.
  - Ubicación de la venta.
  - Cantidades vendidas de cada producto.

- **`PANADERIA/DATA/Bakeryprice.csv`**: Dataset que contiene los precios de los productos disponibles en la panadería.

## Objetivos Específicos

1. **Análisis de Datos**:
   - Identificar patrones de ventas según el día de la semana, hora y ubicación.
   - Detectar productos más vendidos y menos populares.
   - Analizar la relación entre los precios y las cantidades vendidas.

2. **Visualización de Datos**:
   - Crear gráficos que representen las tendencias de ventas.
   - Mostrar la distribución de ingresos por producto y ubicación.

3. **Sistema Accionable**:
   - Proponer estrategias basadas en los datos analizados, como promociones, ajustes de precios o cambios en el inventario.

## Herramientas Utilizadas

- **Python**: Para el análisis y manipulación de datos.
- **Pandas**: Para la limpieza y transformación de datos.
- **Matplotlib y Seaborn**: Para la visualización de datos.
- **Jupyter Notebook**: Para documentar y ejecutar el análisis.
